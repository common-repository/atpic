== Description ==

atpic gallery animated with jquery + simplegallery

* [atpic](http://atpic.com/ "atpic")
* [jquery](http://jqyery.com/ "jquery")
* [simplegallery](http://www.dynamicdrive.com/dynamicindex4/simplegallery.htm "simplegallery")
* atpic plugin ([See in action the online demo](http://www.ushcompu.com.ar/2009/06/29/atpic-plugin/ "atpic online demo"))

== Installation ==

1. Upload `atpic` directory content to `/wp-content/plugins/atpic/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Set options in Widgets menu
